Operadores Aritiméticos

Contém os operadores:
+, -, *, /, **

Operadores de Atribuição

Contém os operadores que atribuem valores à uma variável ou constante.:
Exemplos:
=, +=, -=, *=, etc

Operadores de Igualdade

Igualdade estrita:

Compara e define valores estritamente, respeitando não somente ao dado, mas, também, o seu tipo.
Exemplos:
1 === 1; -> TRUE
'1' === 1; -> FALSE (pois o primeiro é um 'string' e o segundo um 'number', e é recomendável usar desta forma)

Igualdade Solta

Nesse caso a comparação e atribuição ignoram o tipo, se importando apenas com o valor.
Exemplos:
1 == 1; -> TRUE
'1' == 1; -> TRUE (Não é recomendável)

Operador Ternário

Operador que trabalha com condicionais.

Operadores Lógicos

No Javascript existem 3 tipos de operadores lógicos:
"AND", "OR" e "NOT".

Operador Lógico "AND"

Sintaxe representada com o símbolo "&&".
Retorna TRUE se os dois operandos forem TRUE.
Exemplos:.
TRUE && TRUE; -> TRUE
FALSE && TRUE -> FALSE

Para uma vaga de emprego, uma pessoa só pode se aplicar se for de idade E possuir carteira de trabalho.

Operador Lógico "OR"

Sintaxe representada com o símbolo "||".
Retorna TRUE se apenas um dos dois operandos forem TRUE.
Exemplos:.
TRUE && TRUE; -> TRUE
FALSE && TRUE -> TRUE

Exemplo real:
Seguindo o mesmo exemplo anterior, desta vez para outra vaga de emprego, uma pessoa só pode se aplicar se for de idade OU possuir carteira de trabalho.

Operador Lógico "NOT"

Sintaxe representada com o símbolo "!".
Retorna TRUE se apenas um dos dois operandos forem TRUE.
Exemplos:.
let podeAplicar = true;
candidatoRecusado = !podeAplicar; -> FALSE

Comparações Não Boleanos

No Javascript temos 2 conceitos para boleanos, o conceito Truthy e Falsy.
Ao realizar comparações (and e or) o Javascript observara os dados seguindo os conceitos descritos para poder atribuir um valor boleano.

Em Falsy podemos ter os seguintes valores:
undefined, null, 0, false, '', NaN (not a number).

Em Truthy podemos ter todos os valores que não estão listados no Falsy.
Exemplos:
false || 'Rodrigo'; -> Retorna: 'Rodrigo'
null || 3; -> Retorna: 3
NaN || 4 || 'Rodrigo'; -> Retorna: 4
