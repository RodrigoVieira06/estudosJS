Aula 7 - Funções

Abrir o console para conferir dados inseridos no Script

Um padrão de nomenclatura para uma função é um Verbo + Substantivo em camelcase
Por exemplo:
function resetaCor()
function alterarTamanho()
E etc...

Código:
let corSite = "vermelho";

function resetaCor(cor,tonalidade){
corSite = cor + " " +tonalidade;
};

console.log(corSite);
resetaCor("preto","claro");
console.log(corSite);
