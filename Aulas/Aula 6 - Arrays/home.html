Aula 6 - Arrays

Abrir o console para conferir dados inseridos no Script

Arrays são um conjunto de dados que podem ser acessado por um índice.
Índice é uma posição exata de onde se encontra uma informação (dado) nesse conjunto de informações.

Código:
let familia = [true,45,'Rodrigo',17];

console.log(familia.length);
console.log(familia[0]);
