Aula 8 - Tipos de Funções

Abrir o console para conferir dados inseridos no Script

Em JS existem 2 tipos de funções:
- A que realiza uma tarefa e não devolve nada (sem retorno).
- A que realiza uma tarefa e entrega um dado (com retorno).

Código:
//Realiza uma tarefa, mas não devolve(retorna) algum dado.
function dizerNome(){
console.log('Rodrigo');
}
dizerNome();

//Faz um cálculo ou operação e retorna algum dado.
function MultiplicarPorDois(valor){

return valor * 2;
}
let resultado = MultiplicarPorDois(5);
console.log(resultado);

console.log(MultiplicarPorDois(2));
